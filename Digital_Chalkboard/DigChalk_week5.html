Digital Chalkboard Week 5 "More JSFun"

1. Create two input forms on your page to take user input and launch functions.

2. Create an external JavaScript file where you will write two functions.

3. Load that JS file on your HTML page.

4. FIRST FUNCTION: Write a JavaScript function that asks a trivia question. The function should run from an external JS file. Create a div with the question. The input box accepts the user guess and the button launches the function. Can you also figure out how to launch the function if the user hits enter in the input box instead of clicking the button? The function should allow the user to type an answer into an input box. The function checks to see if the answer is correct, then adds a response to the page to repeat what they guessed and if it is correct.

5. SECOND FUNCTION: Write a JS function that asks the user to input a 5-digit number and reports back to the user if the number is odd or even. Your function should also check to make sure the number is, in fact, a 5-digit integer (not a string and no decimals). How did you do it?
